# WordPress_Complete_Theme-_Plugin_Downloader
Any  WordPress Website  Theme &amp; Plugin Downloader Tools


(![image](https://i.postimg.cc/mg9RCHXt/2025-12-23-160851.png)


```bash
git clone https://github.com/chowdhuryvai/WordPress_Complete_Theme-_Plugin_Downloader.git
cd WordPress_Complete_Theme-_Plugin_Downloader
chmod +x WordPress_Complete_Theme-_Plugin_Downloader.py
```


```bash
Usage example:
Full scan and download:
python WordPress_Complete_Theme-_Plugin_Downloader.py https://example.com --download-all
Just download the plugin:
python WordPress_Complete_Theme-_Plugin_Downloader.py https://news-site.com --only-plugins
Just download the Themes:
python WordPress_Complete_Theme-_Plugin_Downloader.py https://news-site.com --only-themes
Deep Scan (for custom items):
python WordPress_Complete_Theme-_Plugin_Downloader.py https://custom-site.com --deep-scan --download-all
Multiple URLs:
python WordPress_Complete_Theme-_Plugin_Downloader.py --urls news_sites.txt --download-all
Custom output:
python WordPress_Complete_Theme-_Plugin_Downloader.py https://example.com --download-all --output my_backup --threads 5
```


```bash
 Special Features:
Custom Theme Detection - From HTML Analysis

News Site Identification - Content Analysis

Directory Listing Scan - Finds open directories

Smart Retry System - Tries 4 Methods

File Size Limit - Downloads files up to 10MB

Progress Tracking - Shows real-time progress

Complete Reports - Detailed JSON and TXT reports

This tool can download custom themes, news themes, and any plugins from any website with their complete file structure.
```
